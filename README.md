## 🌸 Sobre este proyecto

Este repositorio contiene mi portafolio personal, pensado para reflejar mi perfil como desarrolladora front-end con base en diseño gráfico.

Combina una estética cuidada, animaciones suaves y una estructura de código mantenible para ofrecer una experiencia visual atractiva y profesional.

## 💫 Highlights

- 🎨 Identidad visual personalizada con estilo creativo y detalles pixel-art.
- 🌓 Modo claro/oscuro con design tokens en SCSS.
- 🧩 Componentes reutilizables en Vue 3.
- 📱 Layout responsive para desktop y mobile.
- ⚡ Build rápido y optimizado con Vite.

## 🛠️ Tecnologías utilizadas

- Vue 3 (Composition API)
- Vite
- Sass (SCSS)
- JavaScript (ES Modules)

## 🗂️ Estructura del proyecto

```text
src/
	components/   # Componentes reutilizables (botones, tarjetas, heading, etc.)
	sections/     # Secciones del portafolio (Hero, About, Projects, Skills, etc.)
	data/         # Datos centralizados del contenido
	assets/       # Imágenes, íconos, videos y estilos SCSS
	utils/        # Utilidades compartidas
```

## 🚀 Scripts

```bash
npm run dev      # Inicia entorno de desarrollo
npm run build    # Compila para producción
npm run preview  # Previsualiza el build
```

## 💻 Instalación rápida

```bash
npm install
npm run dev
```
